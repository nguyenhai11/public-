--KIỂU HTML 
sylte Thuộc tính HTML được sử dụng để thêm kiểu vào một phần tử, chẳng hạn như màu sắc phông chữ , kích thước ..

ví dụ :
<!DOCTYPE html>
<html>
<body>
<p>Nguyễn Văn A</p>
<p style="color:red;">Nguyễn Văn B</p>
<p style="color:blue;">Nguyễn Văn C</p>
<p style="font-size:60px;">Nguyễn Văn D</p>
</body>
</html>

--Thuộc tính kiểu HTML:
//Thiết lập kiểu của một phần tử HTML, có thể được thực hiện với style thuộc tính.
style thuộc tính có cú pháp như sau:

<tagname style="property:value;">

==Màu nền :
background-color Thuộc tính CSS xác định màu nền cho phần tử HTML

ví dụ :

--KIỂU HTML 
sylte Thuộc tính HTML được sử dụng để thêm kiểu vào một phần tử, chẳng hạn như màu sắc phông chữ , kích thước ..

ví dụ :
<!DOCTYPE html>
<html>
<body style="background-color:powderblue;">
<h1> Đây là đoạn hội thoại nói về ...</h1>
<p>Nội Dung:</p>
</body>
</html>


--Đặt màu nền cho 2 phần tử khác nhau 

<!DOCTYPE html>
<html>
<body>

<h1 style="background-color:powderblue;">Tiêu đề 1</h1>
<p style="background-color:tomato;">Nội dung:</p>
</body>
</html>

--//Văn bản màu 
color Thuộc tính CSS xác định màu văn bản cho một phần tử HTML
VÍ DỤ :
<!DOCTYPE html>
<html>
<body>
<h1 style="color:blue;">Tiêu đề  </h1>
<p style="color:red;">Văn Bản:</p>

</body>
</html>

--//Phông chữ 
font-family Thuộc tính CSS xác định phông chữ được sử dụng cho một phần tử HTML
Ví dụ :
VÍ DỤ :
<!DOCTYPE html>
<html>
<body>
<h1 style="font-family:verdana;">Tiêu đề </h1>
<p style="font-family:courier;">Văn Bản.</p>

</body>
</html>

--// Cỡ chữ 
VÍ DỤ :
<!DOCTYPE html>
<html>
<body>
<h1 style="font-size:300%;">Tiêu đề </h1>
<p style="font-size:120%;">Văn bản.</p>
</body>
</html>

--// Căn chỉnh văn bản 
text-align Thuộc tính CSS xác định căn chỉnh văn bản theo chiều ngang của một phần tử HTML
 
VÍ DỤ

<!DOCTYPE html>
<html>
<body>
<h1 style="text-align:center;">Tiêu đề</h1>
<p style="text-align:center;">Văn Bản</p>

</body>
</html>

TÚM tắt chương 
.Sử dụng stylethuộc tính để tạo kiểu các phần tử HTML
.Sử dụng background-colorcho màu nền
.Sử dụng colorcho màu văn bản
.Sử dụng font-familycho phông chữ văn bản
.Sử dụng font-sizecho các kích thước văn bản
.Sử dụng text-alignđể căn chỉnh văn bản
